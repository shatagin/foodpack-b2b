# FoodPack Project
Веб-сайт разработан в рамках эксплуатационной практики в университете в компании по производству пищевой упаковки.

В процессе прохождения преддипломной практики сайт был доработан путём автоматизации и оптимизации бизнес-процесса обработки заявок. 

Технолоческий стек
- Python 3.14
- Django 5.x
- PostgreSQL 14
- HTML5
- CSS3
- JavaScript
- Gunicorn
- Nginx
- Ubuntu Server 22.04
- Git / GitHub
