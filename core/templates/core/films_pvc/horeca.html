MC / MB для сектора HoReCa

Пищевая ПВХ-плёнка для упаковки готовых блюд, полуфабрикатов и свежих продуктов в ресторанах, кафе и кейтеринге.

Область применения

Используется на профессиональной кухне для упаковки гастроёмкостей, лотков, подносов, порционных и порезанных продуктов.

Примерные характеристики

  • Толщина: по запросу (типичный диапазон 8–12 мкм)
  • Ширина рулона: варианты под оборудование и ручную работу
  • Совместимость с оборудованием HoReCa

Конкретные параметры (толщина, ширина, намотка, тип основы) согласуются индивидуально под оборудование и задачи клиента.
